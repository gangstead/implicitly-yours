## Building IDE files
### Eclipse
This project includes the plugin to generate Eclipse project files.  From the console:
```
> sbt eclipse
```
Then in Eclipse `File > Import > Existing Projects` and find this project.

### IDEA
Intellij IDEA can import SBT projects natively.
